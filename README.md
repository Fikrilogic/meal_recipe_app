# Meal Recipe App

A Recipe App for Android, build with Native Kotlin & Jetpack Compose

# Feature
 - Home
 - Catalog Recipe By Category
 - Catalog Recipe By Area
 - Recipe Favorite
 - Detail of Recipe
 - Add Recipe To Favorite
